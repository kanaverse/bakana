CrisprNormalizationState

CrisprNormalizationState

This step performs normalization and log-transformation on the QC-filtered CRISPR count matrix from the CellFilteringState. It wraps the normalizeCounts functions from scran.js.

Methods not documented here are not part of the stable API and should not be used by applications.
